# Fachowiec Pro

Fachowiec Pro to zaawansowana aplikacja typu PWA (Progressive Web App) stworzona z myślą o profesjonalistach świadczących usługi, takich jak remonty, naprawy czy instalacje. Aplikacja umożliwia kompleksowe zarządzanie biznesem z poziomu urządzenia mobilnego lub komputera.

## Funkcjonalności

*   **Pulpit (Dashboard):** Podsumowanie finansowe, statusy zleceń i ostatnia aktywność.
*   **Klienci:** Baza kontrahentów z szybkim dostępem do kontaktu (telefon/SMS).
*   **Zlecenia:** Zarządzanie procesem realizacji usług, statusy, przypisywanie do klientów.
*   **Kosztorysy:** Tworzenie wycen i generowanie plików PDF.
*   **Raporty:** Podsumowania przychodów i eksport danych do CSV.
*   **Mobile-First:** Responsywny interfejs dostosowany do smartfonów (dolna nawigacja) i desktopów (boczny pasek).

## Uruchomienie projektu

Aby uruchomić aplikację lokalnie, wykonaj następujące kroki:

1.  **Instalacja zależności:**
    ```bash
    npm install
    ```

2.  **Uruchomienie serwera deweloperskiego:**
    ```bash
    npm run dev
    ```
    Aplikacja będzie dostępna pod adresem, który wyświetli się w terminalu (zazwyczaj `http://localhost:5173`).

## Technologie

*   React + Vite
*   Vanilla CSS (zmienne CSS)
*   React Router (nawigacja)
*   Recharts (wykresy)
*   jsPDF (generowanie PDF)
*   Lucide React (ikony)
*   Context API (zarządzanie stanem)
